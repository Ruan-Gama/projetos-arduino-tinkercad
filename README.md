# 🛠️ Portfólio de Automação & Sistemas Embarcados

Este repositório reúne protótipos de hardware e software desenvolvidos com **Arduino** e simulados no **Tinkercad**. [cite_start]O objetivo é demonstrar a aplicação prática de lógica de programação (C/C++), eletrônica básica e controle de periféricos[cite: 57, 60, 65].

## 🚀 Projetos em Destaque

### 1. 🛗 Elevador Funcional
* **Descrição**: Protótipo de um sistema de controle de elevador para múltiplos andares.
* **Destaque Técnico**: Implementação de lógica de estados para gerenciar a posição atual da cabine em relação às chamadas solicitadas, utilizando sensores de pavimento e controle de motor.
* **Localização**: `/elevador-funcional`.

### 2. ⏱️ Cronômetro Digital
* **Descrição**: Sistema de contagem de tempo preciso com interface de usuário.
* **Destaque Técnico**: Controle de tempo em milissegundos com funcionalidades de iniciar, parar e resetar via botões físicos, com saída de dados em display.
* **Localização**: `/cronometro-digital`.

### 3. 📟 Letreiro Digital (LCD)
* **Descrição**: Interface de comunicação visual para exibição de mensagens dinâmicas.
* **Destaque Técnico**: Manipulação de bibliotecas de hardware para exibição de caracteres em displays LCD 16x2, simulando painéis de informação.
* **Localização**: `/letreiro-digital`.

### 4. 🚧 Portão Eletrônico
* **Descrição**: Automação residencial simulando o funcionamento de um portão automatizado.
* **Destaque Técnico**: Uso de lógica condicional para acionamento de motor e integração com sensores de fim de curso para parada segura.
* **Localização**: `/portao-eletronico`.

### 5. 🚦 Semáforo Inteligente
* **Descrição**: Controle de fluxo de tráfego com temporização programada.
* **Destaque Técnico**: Gestão de sequenciamento lógico de LEDs e temporização precisa para simular cenários de sinalização urbana.
* **Localização**: `/Semaforo`.
